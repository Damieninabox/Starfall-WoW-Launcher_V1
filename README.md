# Starfall WoW Launcher

Custom desktop launcher for a private World of Warcraft Cataclysm 4.3.4 server.

See [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) for the spec and [`CLAUDE.md`](CLAUDE.md) for a working summary.

## Quick start

```bash
pnpm install
pnpm tauri dev
```

To run the patcher end-to-end against the local mock server, see [`mock-server/README.md`](mock-server/README.md).
